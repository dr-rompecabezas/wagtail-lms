About the Wagtail LMS Example

This is a demonstration project for the wagtail-lms package.

Getting Started

  1. Create a superuser: python example_project/manage.py createsuperuser
  2. Login to Wagtail Admin
  3. Create a Course Page under the home page
  4. Add lesson children to the course:
    • H5P Lesson Page — upload an H5P activity via LMS → H5P Activities first, then add it to the lesson body
    • SCORM Lesson Page — upload a SCORM package via LMS → SCORM Packages first, then assign it to the lesson
  5. Enroll in the course as a student and test completion tracking

Quick Links

  • Wagtail Admin — manage pages, SCORM packages, H5P activities, enrollments, and attempts
  • Django Admin — manage models directly
